### [905. 区间选点 - AcWing题库](https://www.acwing.com/problem/content/907/)

给定 $N$ 个闭区间 $[a_i,b_i]$，请你在数轴上选择尽量少的点，使得每个区间内至少包含一个选出的点。

输出选择的点的最小数量。

位于区间端点上的点也算作区间内。

#### 输入格式

第一行包含整数 $N$，表示区间数。

接下来 $N$ 行，每行包含两个整数 $a_i,b_i$，表示一个区间的两个端点。

#### 输出格式

输出一个整数，表示所需的点的最小数量。

#### 数据范围

$1 \leq N \leq 10^5,$
$−10^9 \leq a_i \leq b_i \leq 10^9$

#### 输入样例：

```
3
-1 1
2 4
3 5
```

#### 输出样例：

```
2
```
