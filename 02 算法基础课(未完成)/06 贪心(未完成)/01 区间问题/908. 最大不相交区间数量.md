### [908. 最大不相交区间数量 - AcWing题库](https://www.acwing.com/problem/content/910/)

给定 $N$ 个闭区间 $[a_i,b_i]$，请你在数轴上选择若干区间，使得选中的区间之间互不相交（包括端点）。

输出可选取区间的最大数量。

#### 输入格式

第一行包含整数 $N$，表示区间数。

接下来 $N$ 行，每行包含两个整数 $a_i,b_i$，表示一个区间的两个端点。

#### 输出格式

输出一个整数，表示可选取区间的最大数量。

#### 数据范围

$1 \leq N \leq 10^5,$
$−10^9 \leq a_i \leq b_i \leq 10^9$

#### 输入样例：

```
3
-1 1
2 4
3 5
```

#### 输出样例：

```
2
```
