### [885. 求组合数 I - AcWing题库](https://www.acwing.com/problem/content/887/)

给定 $n$ 组询问，每组询问给定两个整数 $a,b$，请你输出 $C_a^b~mod~(10^9+7)$ 的值。

#### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含一组 $a$ 和 $b$。

#### 输出格式

共 $n$ 行，每行输出一个询问的解。

#### 数据范围

$1 \leq n \leq 10000,$
$1 \leq b \leq a \leq 2000$

#### 输入样例：

```
3
3 1
5 3
2 2
```

#### 输出样例：

```
3
10
1
```
